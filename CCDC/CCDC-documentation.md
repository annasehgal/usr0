# CCDC Tools & Best Practices
## Good injects: 
- Formatting
- Description of images/items provided
- Show you did it rather than saying you did it (proof)
- Detailed inject submission

## Important report
- Check formatting for latex
- provide full evidence (full installation of installation)
- show proof of completion through full process
- use report template provided (look for it) 
- dont just type commands in the report, show your outputs running them in screenshots

## Injects Best Practice:
1. Screenshots (good screenshots telling a story)
- Caption all your screenshots
- Aim for 50% graphics, 50% text
- Give full context

2. Writing (for audience, and presentation)
- Dont lie, say what you did
- Work with typos

## Stater best pratice
- change all default credentials, ensure all accounts have set passwords and not accessible with default password/no password.
